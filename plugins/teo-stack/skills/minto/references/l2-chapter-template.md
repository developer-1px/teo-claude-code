## 포맷 고정 규칙 (1.1 레퍼런스에서 추출)

### 표

1. 컬럼 = `요소 | 자연스런 의문 | 원문 | 구체화 | 순서` 5개. 순서 고정.
2. 요소 기호:
   - 첫 행 = `**P**` (볼드), 의문 = `—`
   - 중간 = `D` (볼드 없음)
   - 마지막 = `**P'**` (볼드), 의문 = `(삼단논법 착지)` 또는 `(귀납 착지)` 같은 메타 라벨
3. 원문 = 사용자 말 그대로. 정제·재작성 금지. 파편 → 습니다체 완성 문장 변환만 허용.
4. 순서 컬럼 형식:
   - 첫/마지막 P/P' = `{연역|귀납} · {축} (선언|착지 — {한 줄})`
   - 중간 D = `{연역|귀납} · {대전제|소전제|반박 차단}` 또는 `{연역|귀납} · 정도 N ({근거} — {무게})`
   - 두 차원(추론 종류 + 축) 병기 필수.
5. 연역이면 D 행은 삼단논법 형태(대전제 → 소전제 → 결론/반박 차단). 귀납이면 증거 무게 순(가벼운 → 중간 → 무거운 → 도피처 차단).

### 낭독 블록 (표 직후 `>` blockquote)

1. 1줄 = `> **{헤딩 텍스트 그대로}**` (헤딩 볼드)
2. 2줄 = `>` (빈 blockquote 줄, markdown 문단 분리용 — 무조건)
3. 3줄 = D 원문들을 `**("{의문}")**`로 이어붙인 한 호흡 + P' 원문 마무리
   - 첫 D는 의문 생략 (P → 첫 D 자동 연결)
   - 빈 D 자리 = `**("{의문}")**`만 남기고 원문 건너뛰기
4. 낭독 체크의 목적 = 청중이 떠올릴 자연스런 의문을 추출해 표의 의문 컬럼에 박는 것. ✓/⚠ 채점 아님.

### 의문 톤

- ❌ 적대적 반문 ("왜 단정하시죠?")
- ✅ 자연스런 호기심 ("정확히 뭘 의미하나요? 설정으로 고정할 수 있지 않나요?")
